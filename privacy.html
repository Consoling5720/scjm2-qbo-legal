Privacy Policy

Application: SCJM2 QuickBooks Journal Entry Integration
Owner/Operator: SCJM2, LLC
Last updated: July 9, 2026

This document is plain-English boilerplate provided for transparency and to satisfy platform requirements. It is not legal advice.

1. Scope

This application is a private, internal tool. It has no public users, no customers, and no user accounts other than those belonging to SCJM2, LLC. It collects no information from any member of the public.

2. What Data Is Accessed

Using Intuit's authorized QuickBooks Online API, the application reads and writes only within SCJM2, LLC's own QuickBooks company file. Specifically, it reads chart-of-accounts and location/class metadata needed to construct valid entries, and it writes journal entries that a human has explicitly approved.

The application also processes SCJM2, LLC's own internal payroll summary and franchise fee records. These are aggregate general-ledger totals by account and location. They contain no personally identifiable information.

3. How Data Is Stored

Data is stored on infrastructure controlled solely by SCJM2, LLC. Credentials and API tokens are held in an encrypted secrets store and are never written to disk in plaintext, logged, or transmitted anywhere except to Intuit's own API endpoints over HTTPS. The application is not reachable from the public internet.

4. Data Sharing

No data is sold. No data is rented. No data is shared with, disclosed to, or transferred to any third party for any purpose. There are no advertisers, analytics providers, data brokers, or subprocessors. The only external service the application communicates with is Intuit's QuickBooks Online API, and only to service SCJM2, LLC's own accounting records.

5. Data Retention

Staged and posted entry records are retained on SCJM2, LLC's systems for internal audit purposes. Access tokens are retained only as long as the QuickBooks connection remains active and are destroyed on disconnection.

6. Disconnection

SCJM2, LLC may revoke the application's QuickBooks access at any time from within QuickBooks Online or from the application itself. Upon disconnection, all stored Intuit tokens are deleted and the application can no longer access any QuickBooks data.

7. Security

The application runs on private, non-public infrastructure with outbound-only network access. All communication with Intuit occurs over TLS. Human approval is required before any write to QuickBooks Online occurs.

8. Changes

Any material change to this policy will be reflected here with an updated date.
